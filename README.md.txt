# IoT Health Monitoring System

An Arduino-based project to monitor heart rate, temperature, and humidity in real-time using DHT11 and Pulse sensor.

## Components Used
- Arduino UNO
- DHT11 Sensor
- Pulse Sensor
- LCD 16x2
- Jumper Wires

## Features
- Real-time monitoring
- Serial data display
- Optional cloud dashboard (ThingSpeak)

## How to Run
1. Upload the code in Arduino IDE
2. Connect the sensors as per circuit diagram
3. Open Serial Monitor to view data
